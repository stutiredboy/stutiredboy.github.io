网易游戏 MySQL-MongoDB 运维及 DBA 招聘

2017-11-15 22:51 netease / dba / mysql / mongodb

以下岗位部门均属于网易在线游戏事业部。 其中 MongoDB 运维及运维开发工程师可选工作地点为广州或杭州。

相关数据库产品均服务于如《荒野行动》、《梦幻西游》、《大话西游》、《阴阳师》、《我的世界》、CC等网易游戏海量用户业务。 如果需要也有更深入了解的，也可以给我留言或者微信（二维码戳我）私我。同时欢迎推荐及转发。

岗位一：MySQL DBA 点我投递简历

工作职责

  1. 参与内部MySQL SaaS开发和维护工作；
  2. 优化数据库，提高数据处理效率；
  3. 负责数据库新技术的调研和测试；
  4. 负责处理故障，服务请求，运维环境优化；
  5. 和开发团队及其他部门进行沟通协作。

职位要求

  1. 精通MySQL等数据库系统使用，精通SQL语句和优化；
  2. 熟悉Linux系统操作；
  3. 了解网络，有一定的故障排查和处理能力；
  4. 熟悉shell脚本编程，会Python、Java者优先；
  5. 有大型数据库和大规模服务器管理经验优先；

岗位二：MongoDB 运维工程师 点我投递简历

职位描述：

  1. 负责 MongoDB 等数据库平台的配置管理、流程管理；
  2. 响应用户需求，提供协助或技术解决方案，配合业务完成数据库集群的部署、升级、扩容、缩容等；
  3. 通过全方位、立体化监控系统快速发现和处理故障；
  4. 配合优化数据库服务运维规范、工作流程、应急预案等，确保任何突发情况都能高效响应；
  5. 汇总用户需求，并设计技术需求方案，促进产品发展；

职位要求：

  1. 熟悉 Linux 系统以及工具命令等使用，熟悉主流开源软件的配置及调优，有底层基础技术研发经历的优先；
  2. 熟悉至少一种程序设计语言，有 Shell、Python 等脚本编程语言经验者优先；
  3. 对计算机硬件设备、网络设备有一定了解，熟悉 TCP/IP 以及具有丰富的网络知识；
  4. 熟悉 MongoDB 数据库的日常使用；
  5. 掌握 MongoDB 的管理及构建，包括但不限于集群构建、性能调优、备份恢复；
  6. 具备良好的沟通表达能力和服务意识，较强的团队合作意识，快速处理突发事件的能力；
  7. 有大型 MongoDB 商用部署实施及运营经验者优先，有 OCP 认证者优先。

岗位三：运维开发工程师（数据库平台）点我投递简历

职位描述：

  1. 负责内部数据库相关产品的管理平台、自动化流程开发工作；
  2. 与产品、PM、运维同学协作，积极高效推动产品开发；
  3. 与用户及其他部门进行沟通协作；

职位要求：

  1. 1 年或以上前后端开发工作经验；
  2. 对开发有浓厚兴趣，HTML/CSS/JS 基础扎实，熟悉Angulajs、Reactjs、Vue.js等MVVM框架中的一种；
  3. 有至少一种 server 端开发经验，熟悉 Python、PHP 语言开发者优先；
  4. 至少一种熟悉Web开发框架（如Django/Flask/Tornado）；
  5. 对 HTTP 协议、网页性能优化有一定的理解；
  6. 熟悉 MySQL、MongoDB、Redis 等数据库应用开发优先；
  7. 能够编写高扩展、易维护的代码，熟练且规范使用代码管理工具；

岗位四：高级运维工程师（DNS方向） 点我投递简历

职位描述：

  1. 负责 DNS 及配套系统部署、维护及运维环境、服务优化；
  2. 日常系统运维、服务请求、故障处理等技术支持；
  3. 网络、系统等维护配套系统部署、维护；
  4. 和项目开发团队及其他部门进行沟通协作；
  5. 负责新技术的调研和测试。

职位要求：

  1. 熟悉 Linux 系统操作；
  2. 熟悉 shell 脚本语言编程，会 Python 者优先；
  3. 了解 TCP/IP 网络，有一定的故障排查和处理能力；
  4. 熟悉 DNS 协议、BIND，有相关运维经验，并能够快速实施部署、配置和排障；
